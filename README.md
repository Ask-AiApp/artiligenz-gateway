# Artiligenz Gateway

The **Artiligenz Gateway** is the central landing hub for the Artiligenz AI ecosystem.  
It’s the entry point into your AI “universe”, connecting users to:

- **AI Directory** – live interactive map of the AI ecosystem
- **AI Information Hub** – news, insights, and breakthroughs
- Future Artiligenz apps and experiences

---

## Project Info

- **Live site**: https://artiligenz.com  
- **Tech stack**: Vite + React + TypeScript + Tailwind CSS + shadcn-ui
- **Backend / data**: Supabase

---

## Tech Stack

This project is built with:

- **[Vite](https://vitejs.dev/)** – bundler & dev server
- **[React](https://react.dev/)** – UI library
- **[TypeScript](https://www.typescriptlang.org/)** – typed JavaScript
- **[Tailwind CSS](https://tailwindcss.com/)** – utility-first styling
- **[shadcn-ui](https://ui.shadcn.com/)** – component library built on Radix
- **[Supabase](https://supabase.com/)** – database, auth, and APIs

> Note: `vite` is installed as a **dependency** (not only a devDependency) so that Render can run `vite build` during deployment.
